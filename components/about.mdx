The About component presents personal information and technical expertise through an elegant two-part layout: a text introduction and a grid of service cards.

## Overview

The component features:
- Left-aligned introduction with vertical accent line
- Animated text reveal on scroll
- Grid of service cards with hover effects
- Decorative background gradients
- Full dark mode support

## Component Structure

```jsx About.jsx
import React from "react";
import { motion } from "framer-motion";
import { styles } from '@/styles';
import { about_page_text, services } from '@/constants';
import { SectionWrapper } from '@/hooks';
import ServiceCard from "./ServiceCard";

function About() {
  return (
    <div className="relative px-4 sm:px-6 lg:px-8 
      bg-white/50 dark:bg-zinc-950/30 backdrop-blur-sm 
      pt-12 pb-20 transition-colors duration-300">
      {/* Decorative backgrounds */}
      {/* About Me section */}
      {/* Technical Expertise section */}
    </div>
  );
}

export default SectionWrapper(About, "about");
```

## Layout Sections

### 1. Introduction Section

Features a vertical accent line and animated text:

```jsx
<div className="flex gap-8 mb-12">
  {/* Vertical Accent Line */}
  <motion.div 
    initial={{ scaleY: 0 }}
    whileInView={{ scaleY: 1 }}
    transition={{ duration: 1, ease: "easeOut" }}
    viewport={{ once: true }}
    className="hidden md:block w-1.5 bg-gradient-to-b 
      from-navy-500 via-navy-600 to-transparent 
      dark:from-blue-500 dark:via-blue-600 dark:to-transparent

  <div className="flex-1">
    <motion.div

    <motion.p
```

<Info>
The vertical line animates from top to bottom using `scaleY` transform for smooth performance.
</Info>

### 3. Service Cards Grid

```jsx
<div className="grid grid-cols-1 sm:grid-cols-2 lg:grid-cols-4 gap-6">
  {services.map((service, index) => (
    <motion.div 
      key={service.title}
      initial={{ opacity: 0, y: 40 }}
      whileInView={{ opacity: 1, y: 0 }}
      viewport={{ once: true }}
      transition={{ delay: index * 0.15, duration: 0.6 }}
    >
      <ServiceCard index={index} {...service} />
    </motion.div>
  ))}
</div>
```

<Note>
Cards animate in with staggered delays (0.15s apart) for a cascading effect.
</Note>

## SectionWrapper HOC

The component uses `SectionWrapper` for:
- Section ID for navigation
- Scroll-based animations
- Consistent padding and margins
- Viewport detection

```javascript
export default SectionWrapper(About, "about");
```
